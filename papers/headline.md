## 4. 方法 (Methodology) [2.5页]

### 4.1 问题定义
- 输入: 低质量路面图像 $I_{LR}$
- 输出: 裂缝分割掩码 $M_{pred}$
- 两阶段流程: $I_{LR} \xrightarrow{SRCNN} I_{SR} \xrightarrow{U-Net} M_{pred}$

### 4.2 数据预处理与智能降质

#### 4.2.1 降质模型
$$I_{LR} = D(I_{HR}, \theta)$$

降质类型:
- 高斯模糊: $k \in [3,9], \sigma \in [0.5, 3.0]$
- 下采样: $scale \in [2, 4]$
- JPEG压缩: $quality \in [30, 70]$
- 组合降质

#### 4.2.2 Mask引导的智能降质
```
裂缝区域: 强模糊 (k=[7,13], σ=[2.0,4.0])
背景区域: 弱模糊 (k=[3,5], σ=[0.3,1.0])
边缘平滑: 高斯模糊mask边缘实现无缝融合
```

### 4.3 SRCNN图像重建

#### 4.3.1 基础SRCNN架构 (3层)
```
Conv1: 9×9, 64 filters, ReLU
Conv2: 1×1, 32 filters, ReLU
Conv3: 5×5, 3 filters (output)
```
参数量: 57,184

#### 4.3.2 改进SRCNN (7层, 全3x3)
```
Conv1-6: 3×3, 64 filters, ReLU
Conv7: 3×3, 3 filters (output)
残差连接: output = learned + input
```
参数量: 219,075

#### 4.3.3 损失函数
$$L_{SR} = \frac{1}{N}\sum_{i=1}^{N}\|f_{SRCNN}(I_{LR}^{(i)}) - I_{HR}^{(i)}\|_2^2$$

### 4.4 U-Net裂缝分割

#### 4.4.1 网络架构
- 编码器: 4层下采样 (64→128→256→512)
- 解码器: 4层上采样 + 跳跃连接
- 输出: 单通道分割图

#### 4.4.2 损失函数
$$L_{seg} = 0.5 \cdot L_{BCE} + 0.5 \cdot L_{Dice}$$

BCE加入正样本权重 $w_p=5.0$ 解决类别不平衡。

## 6. 结论 (Conclusion) [0.5页]

### 6.1 总结
- 提出SRCNN+U-Net两阶段低质量裂缝检测框架
- 在CRACK500数据集上达到87.28% IoU
- 相比传统插值方法提升135.9%

### 6.2 局限性
- 两阶段独立训练，未实现端到端优化
- SRCNN对极细裂缝恢复能力有限

### 6.3 未来工作
- 端到端联合训练
- 引入Transformer架构
- 扩散模型数据增强

数据里藏着三个可以极大提升课程论文逼格的“洞察（Insights）”，强烈建议在第5章单独开小节讨论：

关于 BatchNorm 导致性能下降的深度分析：
你的实验中带有BN的改进版（24.10 dB）远差于不带BN的版本（30.13 dB）。你可以在论文中指出：在超分辨率这种“图像到图像”的底层重建任务中，网络的目的是恢复绝对的像素强度和颜色对比度。BatchNorm 会对每个Batch的特征进行强行归一化，破坏了图像原有的对比度信息并抹杀了高频边缘细节。著名超分网络EDSR（你的参考文献）的核心贡献之一就是去除了BN层。你的实验完美印证了这一底层视觉任务的通用规律。

PSNR 最高 ≠ 下游分割表现最好 (Task-Driven 悖论)：
这是一个非常高阶的学术发现。你的实验显示：

Improved SRCNN all 3x3：PSNR最高（30.81 dB），但 IoU 略低（87.21%）。

Improved SRCNN (含大卷积核)：PSNR略低（30.09 dB），但 IoU 最高（87.28%）。
你可以在论文中讨论：PSNR是基于均方误差（MSE）计算的，它倾向于全局像素的过度平滑。虽然全3x3卷积核拟合出了全局误差最小的图像（PSNR更高），但包含大感受野（9x9）的网络保留了更多有助于U-Net进行逻辑推理的结构性高频特征。这说明面向人类视觉的评价指标（PSNR）与面向下游计算机视觉任务（语义分割）的最优特征之间存在偏差。

插值方法的“完全失效”：
论文中务必放上你生成的 No Restore、Bilinear、Bicubic 完全一样的表格结果。并解释其原因：由于输入已经缺失了高频信息，数学插值只是把一个模糊的像素变成了四个模糊的像素，对于解决严重类别不平衡的裂缝分割毫无帮助。
