# 4. 实验结果与分析

## 4.1 实验设置

### 4.1.1 数据集

本文实验基于CRACK500数据集[2]构建。CRACK500是目前路面裂缝检测领域广泛使用的基准数据集之一，包含多种真实路面环境下的裂缝图像。本文使用的数据集图像尺寸为640×360（或360×640），在预处理阶段统一调整为256×256像素，并应用第3.2节所述的智能降质策略生成低质量训练样本。数据集划分如下：

| 数据集 | 图像数量 | 比例 |
|--------|----------|------|
| 训练集 | 2,712 | 80% |
| 验证集 | 316 | 10% |
| 测试集 | 336 | 10% |
| **总计** | **3,364** | 100% |

统计显示，训练集中裂缝像素占比约为6%，存在显著的类别不平衡问题。

### 4.1.2 实验环境与参数

**硬件环境**：NVIDIA GPU，CUDA加速。

**软件框架**：PyTorch深度学习框架。

**训练参数**：

| 参数 | SRCNN | U-Net |
|------|-------|-------|
| 优化器 | Adam | Adam |
| 学习率 | $10^{-4}$ | $10^{-4}$ |
| 批量大小 | 16 | 8 |
| 训练轮数 | 100 | 100 |
| 损失函数 | MSE | BCE + Dice |
| 正样本权重 | - | 5.0 |

**两阶段训练流程**：首先使用图像对$(I_{LR}, I_{HR})$训练SRCNN网络；然后使用训练好的SRCNN对训练集图像进行重建，生成超分辨率图像用于训练U-Net分割网络。

### 4.1.3 评价指标

**超分辨率评价指标**：采用峰值信噪比（PSNR）和结构相似性（SSIM）衡量重建质量。

**分割评价指标**：采用交并比（IoU）、Dice系数和像素精度（Accuracy）评估分割性能。其中IoU为最主要指标，定义为预测裂缝区域与真实裂缝区域的交集与并集之比。

## 4.2 超分辨率重建实验

### 4.2.1 SRCNN模型对比

本文基于Dong等[3]提出的SRCNN架构，对比了三种不同深度的重建网络性能：

| 模型 | PSNR (dB) | SSIM | 感受野 |
|------|-----------|------|--------|
| Basic SRCNN (3层) | 27.41 | 0.9076 | 13×13 |
| Improved SRCNN (5层) | 30.09 | 0.9482 | 19×19 |
| Improved SRCNN (7层全3×3) | 30.81 | 0.9545 | 15×15 |

随着网络深度增加，PSNR从27.41 dB逐步提升至30.81 dB，SSIM从0.9076提升至0.9545，表明增加深度和采用小卷积核策略均能有效提升重建质量。详细的消融分析见4.4.2节。

### 4.2.2 BatchNorm层的影响

消融实验对比了5层网络在含/不含BatchNorm层下的表现：

| 网络配置 | PSNR (dB) | SSIM |
|----------|-----------|------|
| 5层 + BatchNorm | 24.10 | 0.8400 |
| 5层 无BatchNorm | **30.09** | **0.9482** |

含BN的版本PSNR骤降至24.10 dB，移除后恢复至30.09 dB，差距达5.99 dB。原因分析见4.4.2节。

## 4.3 裂缝分割实验

### 4.3.1 主实验结果

本文采用Ronneberger等[4]提出的U-Net作为分割网络，系统对比了不同预处理方法对裂缝分割性能的影响：

| 方法 | IoU (%) | Dice (%) | Accuracy (%) |
|------|---------|----------|--------------|
| Bilinear + U-Net | 36.96 | 49.41 | 95.70 |
| Bicubic + U-Net | 36.96 | 49.41 | 95.70 |
| Original HR + U-Net | 59.55 | 72.87 | 97.20 |
| Basic SRCNN + U-Net | 86.84 | 92.47 | 99.44 |
| **Improved SRCNN + U-Net** | **87.28** | **92.67** | **99.47** |
| Improved 3×3 + U-Net | 87.21 | 92.61 | 99.47 |

## 4.4 分析与讨论

### 4.4.1 学习型超分辨率对裂缝检测的必要性

实验结果表明，传统插值方法与学习型超分辨率之间存在本质差距。双线性和双三次插值方法的IoU仅为36.96%，相比直接使用原始高分辨率图像（IoU 59.55%）反而下降了38.0%，证明传统插值在严重降质场景下完全失效——其仅基于周围像素的数学平滑，无法恢复丢失的边缘信息。

引入SRCNN后，IoU从36.96%跃升至86.84%以上，相对提升超过135%。更值得注意的是，使用SRCNN重建图像训练的U-Net甚至优于直接使用原始高分辨率图像训练的U-Net（IoU 86.84%~87.28% vs 59.55%）。这说明SRCNN的端到端学习机制能够重建利于分割的判别性特征，而非单纯追求像素相似度。对于低质量路面图像，深度学习驱动的超分辨率重建是机器视觉模型提取有效特征不可或缺的前置步骤。

### 4.4.2 SRCNN网络结构的消融分析

**BatchNorm层的负面影响**：含BN的5层网络PSNR骤降至24.10 dB，甚至低于基础3层版（27.41 dB），而移除BN后回升至30.09 dB。这说明在裂缝重建这种强调极细微对比度突变的任务中，BN层的特征归一化不仅无法加速训练，反而抹杀了关键的结构性高频特征，与Lim等[5]在EDSR中的设计原则一致。

### 4.4.3 重建指标与下游任务指标的非正相关性

本研究的一个核心发现是：面向人类视觉评价的超分物理指标（PSNR）与面向机器视觉的下游任务指标（IoU）之间存在非严格的正相关性，如图5所示。

![图 5 PSNR与IoU的非正相关性](figures/psnr_vs_iou.png)

| 模型 | PSNR (dB) | IoU (%) | 感受野 |
|------|-----------|---------|--------|
| Improved SRCNN (5层) | 30.09 | **87.28** | 19×19 |
| Improved SRCNN (7层) | **30.81** | 87.21 | 15×15 |

7层全3×3模型取得了最高的PSNR（30.81 dB），但IoU（87.21%）却略低于5层模型（30.09 dB PSNR，87.28% IoU）。究其原因，PSNR基于全局均方误差计算，7层全3×3网络为追求全局像素误差最小化，可能在数学上过度平滑了部分极端突变的极细裂缝边缘。而5层模型首尾保留的大卷积核（9×9和5×5）虽然在全局像素拟合上略逊，但其更大的局部感受野（19×19 vs 15×15）更精准地保留了裂缝的宏观结构连续性，为下游U-Net提供了更有效的特征表达。这一发现提示：超分辨率网络应面向具体下游任务设计，而非单纯追求PSNR指标的最大化。

### 4.4.4 类别不平衡问题的应对

裂缝像素在图像中占比仅约6%，严重的类别不平衡是分割任务的主要挑战。本文通过加权BCE损失（为裂缝像素赋予5倍权重）与Dice损失的加权组合来应对，兼顾像素级分类精度和区域级重叠度。最终达到99.47%的像素精度，同时保持了87.28%的IoU，验证了该组合损失策略在极端不平衡场景下的有效性。
